# Bored Tour
Helping people find things to do with their time.

## Setup
1. Install dependencies:
```
npm install
```
2. Setup Postgres database to match settings in `/database/index.js`.
3. Rename `.env.example` to `.env` and fill in the enironment variables.
4. Build scripts:
```
npm run build
```
5. Start the server:
```
npm start
```
6. Open the site:
```
http://localhost:3000
```
